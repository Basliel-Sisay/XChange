# XChange 

A high performance Node.js CLI utility for real time currency conversion with a professional grade HUD

## Overview
Developed as part of Day 9 in the 30 Day Full Stack Challenge, this tool provides a sleek terminal experience for accessing global financial markets. It uses a Promise based interactive dialogue to guide users through conversions across 160+ world currencies

##  Tech Stack
* **Runtime:** Node.js
* **HTTP Client:** Axios
* **Environment:** Native Node.js Env Loader
* **UI:** Xterm-256 ANSI Styling

## File Structure
```text
XChange/
├── node_modules/         # System dependencies (Auto generated)
├── .env                  # Private API credentials
├── .gitignore            # Stealth configuration (File exclusions)
├── LICENSE               # Legal usage permissions
├── README.md             # System documentation and HUD guide
├── converter.js          # Core logic and Stark Financial Interface
├── package-lock.json     # Dependency version lock
└── package.json          # Project manifest and start scripts
```

##  Quick Start

1. **Clone and Install:**
   ```bash
   git clone  https://github.com/Basliel-Sisay/XChange.git
   npm install
   ```
2. **Setup Environment:**
      
   Add your api and url to the .env file

3. **Execute:**

   ```bash
   npm start
   ```
